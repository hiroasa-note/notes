広浅ノートとは？

「広浅ノート」は、広く浅く、気になったものを記録していく雑記ブログです。

AI、Web制作、教育、ゲーム、本、ガジェット、旅行。

テーマはかなり雑多です。

でも、自分の中では全部つながっています。

昔から、一つのことを深く掘り続けるより、色んな分野を行ったり来たりする方が性に合っていました。

その時は意味がないように見えても、後から知識同士がつながることがあります。

このサイトは、そんな「広く浅く」の記録です。

なぜ“広く浅く”なのか

“広く浅い”という言葉は、どちらかといえばネガティブに使われがちです。

でも、最近は少し風向きが変わってきた気がしています。

AIや検索技術の進化によって、「全部を暗記する」よりも、

  • 必要な情報へたどり着ける
  • 分野同士をつなげられる
  • 違う知識を組み合わせられる

こういった力の価値が上がっているように感じるからです。

例えば、

  • 歴史の話
  • AI企業の話
  • Web技術の話
  • 教育の話

一見関係ないようでも、考えていくと意外につながっていきます。

このブログでは、そういう「接点」を残していきたいと思っています。

このサイトで扱う予定のテーマ

現在考えているテーマはこんな感じです。

  • AI・LLM
  • Web制作
  • Astroブログ運営
  • GitHub Pages
  • 教育・学習
  • ゲーム
  • ガジェット
  • 本・漫画
  • 日常の気づき
  • 雑記エッセー

かなり自由です。

ただ、「あとで誰かの役に立つかもしれない」という視点だけは、できるだけ残したいと思っています。

Astroブログを選んだ理由

このサイトは、Astroを使って構築しています。

理由はシンプルで、

  • 軽い
  • Markdownで管理しやすい
  • GitHub Pagesと相性がいい
  • HTMLを直接触れる
  • 構造を自分で理解しやすい

と感じたからです。

最近は便利なサービスも多いですが、自分で仕組みを理解しながら作る方が好きでした。

まだ試行錯誤中ですが、その過程も含めて記録していく予定です。

完璧さより、まず記録

このサイトでは、完璧な記事を書くことを最優先にはしていません。

もちろん、できるだけ一次情報は確認します。

ただ、それ以上に、

「気になったことを残しておく」

ことを重視しています。

あとから読み返した時に、自分の思考ログとして機能するかもしれない。

あるいは、偶然検索でたどり着いた誰かの役に立つかもしれない。

そのくらいの距離感で続けていく予定です。

今後について

まずは小さく更新を続けていきます。

最初は雑多でも、あとからテーマ同士がつながるかもしれません。

広く浅く。

でも、完全に無意味ではない。

そんな記録サイトを目指しています。
